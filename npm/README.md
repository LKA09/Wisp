# Wisp

Local-first coding agent orchestrator for Claude and Codex.

## Install

```sh
npm install -g @lka09/wisp
```

On install, Wisp downloads a prebuilt binary from GitHub Releases for the
current platform. If the binary is unavailable, build from source in the repo.

## Repository

- GitHub: https://github.com/LKA09/Wisp
- Releases: https://github.com/LKA09/Wisp/releases

## Quick start

```sh
wisp init
wisp summon "<your task>"
```
